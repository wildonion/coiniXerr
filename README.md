# Development Setup

* **Install `cargo-watch`:** ```cargo install cargo-watch```

* **Watch _psychoder_ service:** ```cargo watch -x 'run --bin psychoder'```

# Production Setup

* **Build _psychoder_ service:** ```cargo build --bin psychoder --release```
